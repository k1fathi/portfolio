Personal Data Protection Authority, Complaint System

The "KVKK Şikayet Modülü Kılavuzu" is a user guide provided by the Turkish Personal Data Protection Authority (KVKK) to assist individuals in submitting complaints regarding personal data violations. Accessible via the KVKK website or through the e-Government portal, the module allows users to file complaints directly.

Accessing the Complaint Module

  • Visit www.kvkk.gov.tr and select "İlgili Kişi Şikayet Bildirimi" on the homepage.
  • Alternatively, access the module through the e-Government system.

Filing a Complaint

Profile Completion
  • Log in using e-Government credentials.
  • Ensure your profile is 100% complete; incomplete profiles cannot submit complaints.
  • Update personal information, including obtaining a 10-digit address number from https://adres.nvi.gov.tr/VatandasIslemleri/AdresSorgu if necessary.
Submitting a Complaint
  • After completing your profile, navigate to "Şikayetler" and select "Yeni Şikayet Oluştur."
  • Provide detailed information about the complaint, including the data controller's details and the nature of the violation.
  • Attach any supporting documents as evidence.
Review and Submission
  • Review the entered information for accuracy.
  • Submit the complaint; a confirmation message will appear upon successful submission.

The guide emphasizes that only individuals submitting complaints on their own behalf can use the module; representatives or guardians cannot file on behalf of others through this system. Additionally

Vibte AI

Vibte AI is a project focused on building a custom AI model to enhance user engagement and content relevance on a platform (like a website or app). The core idea revolves around creating a model from scratch using TensorFlow, a powerful open-source machine learning library. This approach allows for granular control over the model's architecture and training process, tailoring it specifically to the platform's needs.

The unique aspect of Vibte AI lies in its training methodology: it leverages real-time customer reactions and interactions on the platform. This means the model learns directly from user behavior, such as clicks, likes, shares, comments, and time spent on specific content. This feedback loop creates a dynamic learning environment, allowing the model to continuously improve its understanding of user preferences.

The development process involves defining the model's architecture within TensorFlow, selecting appropriate algorithms, and designing a data pipeline to efficiently process user interaction data. This data is then used to train the model, iteratively adjusting its parameters to optimize performance. By training on real user data, Vibte AI aims to provide highly personalized recommendations, content suggestions, and overall improved user experience. This approach ensures the model stays relevant and adapts to evolving user trends, ultimately driving engagement and satisfaction.
